# perceptron
感知器算法是一种可以直接得到线性判别函数的线性分类方法，它采用最直观的准则，即最小错分样本数，
将错分样本到判别界面的距离之和作为准则，称为感知器准则。表示如下:
$$J ( \boldsymbol { a } ) = \sum _ { \boldsymbol { y } \in Y } - \boldsymbol { a } ^ { T } \boldsymbol { y }$$
